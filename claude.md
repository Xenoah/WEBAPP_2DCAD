## 判断記録

| 日付 | 判断 | 理由 |
|------|------|------|
| 2026-07-03 | DWG ネイティブ対応を見送り、DXF R12 書き出し + R12〜2018 読み込みを互換の柱とする | DWG は非公開バイナリ形式で、合法かつオフラインで完全互換を実現する手段が存在しない。LibreDWG は GPL かつ WASM 化に難があるため Phase 7 で調査扱い |
| 2026-07-03 | 書き出しは R12 (AC1009) 固定 | R2000+ はハンドル・全テーブル・OBJECTS 辞書が必須でボイラープレートが大きい。R12 は最小構成で最も広く開ける。LWPOLYLINE は POLYLINE/VERTEX に変換(bulge 保持)し情報欠損なし |
| 2026-07-03 | コマンドエンジンはジェネレータ方式 | LINE→点→点→… のような対話フローを yield {type:'point',...} で直列に書け、AutoCAD 的なプロンプト遷移・キーワード・キャンセルを一元管理できる |
| 2026-07-03 | フレームワーク不使用（バニラ JS） | PDF エディタ (XNH_WEBAPP_pdf-editor) と同方針。ビルド不要・オフライン重視。描画は全て単一 canvas の即時再描画で React 等の恩恵が薄い |
| 2026-07-03 | 色は ACI (AutoCAD Color Index) を内部表現に採用 | DXF 62 グループコードと1対1対応し往復変換で欠損しない。256=ByLayer |

## 手動QAチェックリスト

### 基本
- [ ] `index.html` を file:// で開いて起動する（外部通信ゼロ）
- [ ] L → クリック2点 → Enter で線分が引ける。右クリックでコマンド繰り返し
- [ ] `10,20` `@50,0` `@30<45` の座標入力が効く。直交 ON で直接距離入力
- [ ] Esc でコマンドキャンセル、描きかけが残らない

### 作図・修正
- [ ] 全作図コマンド（L/C/A/PL/REC/PO/T）でエンティティが作れる
- [ ] M/CO/RO/SC/MI/O/E/X が選択済み・未選択どちらの順でも動く
- [ ] 窓選択（左→右）は内包のみ、交差選択（右→左）は接触も選ばれる

### スナップ・表示
- [ ] 端点・中点・中心・四半円点・交点マーカーが出て、その点に吸着する
- [ ] F8 直交 / F9 スナップ / F7 グリッド / F3 OSNAP / F10 極 が切り替わる
- [ ] ホイールズームがカーソル中心、中ドラッグでパン、Z→E で全体表示

### 画層・プロパティ
- [ ] 画層追加・カレント変更・非表示・ロック・色変更が描画に反映される
- [ ] 非表示画層は選択不可、ロック画層は選択できるが修正不可
- [ ] 選択物の画層・色をプロパティパネルから変更できる

### Undo / ファイル
- [ ] 全コマンドが Undo / Redo で復元される（コマンド単位）
- [ ] DXF 書き出し → AutoCAD / DWG TrueView / ODA Viewer で開ける
- [ ] 他 CAD の DXF（R12/2000/2018, bulge 付きポリライン含む）が読める
- [ ] 未対応エンティティはスキップ件数がトーストで出る
- [ ] 未保存で閉じようとすると警告が出る
