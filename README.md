# WEBAPP_2DCAD — ブラウザ完結型 2D CAD

AutoCAD ライクな操作体系（コマンドライン駆動・エイリアス・オブジェクトスナップ）を持つ、
**完全オフライン動作**のブラウザ 2D CAD。ビルド不要、`index.html` を開くだけで動作する。

## 設計方針

- **バニラ HTML + CSS + JavaScript**（ビルドツール不要、CDN 非依存、外部通信ゼロ）
- **AutoCAD と同じ操作モデル**
  - 画面下部のコマンドラインが主役。`LINE` / `L`、`CIRCLE` / `C` などのコマンドとエイリアス
  - 座標入力: 絶対 `10,20` / 相対 `@5,5` / 極座標 `@10<45` / 直接距離入力（数値のみ）
  - Enter / Space / 右クリック = 確定・コマンド繰り返し、Esc = キャンセル
  - 名詞→動詞（先に選択して MOVE）と 動詞→名詞（MOVE してから選択）の両対応
  - 窓選択（左→右、青 / 内包）・交差選択（右→左、緑 / 交差）
  - オブジェクトスナップ（端点・中点・中心・四半円点・交点）、直交モード、極トラッキング、グリッドスナップ
  - F3 / F7 / F8 / F9 / F10 のファンクションキートグル
- **図面データはメモリ上のエンティティ配列**。DXF へシリアライズして保存

## ファイル互換性

| 形式 | 読み込み | 書き出し | 備考 |
|------|---------|---------|------|
| DXF (ASCII, R12〜2018) | ✅ 対応エンティティのみ | ✅ R12 形式で出力 | LINE / CIRCLE / ARC / LWPOLYLINE / POLYLINE / TEXT / MTEXT(簡易) / POINT、LAYER テーブル、bulge(円弧付きポリライン) |
| DWG | ❌ | ❌ | **DWG はオートデスク非公開のバイナリ形式のため、純ブラウザ・オフラインでの完全互換は不可能。** 将来 LibreDWG (WASM) の同梱を検討（Phase 7）。当面は AutoCAD / 無償の ODA File Converter 等で DXF に変換して使用する |

> 書き出しを R12 にしている理由: ハンドルテーブル等の必須ボイラープレートが最小で、
> AutoCAD 本体・互換 CAD・ビューアの実装がもっとも広く受け付ける「最大公約数」形式のため。

## 実装済みコマンド（v1）

### 作図
| コマンド | エイリアス | 内容 |
|---|---|---|
| LINE | L | 線分（連続入力、U で1点戻る、C で閉じる） |
| CIRCLE | C | 円（中心→半径 or 直径 D） |
| ARC | A | 円弧（3点） |
| PLINE | PL | ポリライン（C で閉じる） |
| RECTANG | REC | 長方形（2対角点） |
| POINT | PO | 点 |
| TEXT | T, DT | 文字（挿入点→高さ→角度→文字列） |

### 修正
| コマンド | エイリアス | 内容 |
|---|---|---|
| MOVE | M | 移動 |
| COPY | CO, CP | 複写（連続配置） |
| ROTATE | RO | 回転（角度入力 or 点指定） |
| SCALE | SC | 尺度変更 |
| MIRROR | MI | 鏡像（元を残す/削除を選択） |
| OFFSET | O | オフセット（線分・円・円弧） |
| ERASE | E | 削除（Delete キーでも可） |
| EXPLODE | X | ポリライン分解 |

### 表示・ユーティリティ・ファイル
| コマンド | エイリアス | 内容 |
|---|---|---|
| ZOOM | Z | E=図形範囲 / A=全体 / W=窓（ホイール=ズーム、中ドラッグ=画面移動） |
| PAN | P | 画面移動（左ドラッグ、Esc で終了） |
| REGEN | RE | 再描画 |
| UNDO / REDO | U / Ctrl+Z, Ctrl+Y | 元に戻す / やり直し |
| LAYER | LA | 画層パネルへフォーカス |
| DIST | DI | 2点間距離計測 |
| NEW / OPEN / SAVEAS | — | 新規 / DXF を開く / DXF 書き出し |

## 使い方

1. `index.html` をブラウザで開く（サーバ不要、file:// で動作）
2. コマンドラインに `L` と入力して Enter → 始点・次点をクリック → Enter で終了
3. 保存はリボンの「DXF 書き出し」または `SAVEAS`

## 既知の制約（v1）

- DWG 読み書き不可（上記の通り。DXF 経由で運用）
- TRIM / EXTEND / FILLET / CHAMFER / HATCH / BLOCK / 寸法（DIM 系）は未実装（Phase 2〜のロードマップ参照 → claude.md）
- DXF の INSERT（ブロック参照）・HATCH・SPLINE・楕円は読み込み時スキップ（件数を通知）
- 線種は CONTINUOUS / DASHED / CENTER / HIDDEN の4種
- 印刷（PLOT）・ペーパー空間・レイアウトタブは未実装

## ライセンス・依存

外部ライブラリ依存なし（自前実装のみ）。
