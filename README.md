# WEBAPP_2DCAD — ブラウザ完結型 2D CAD

AutoCAD ライクな操作体系（コマンドライン駆動・エイリアス・オブジェクトスナップ）を持つ、
**完全オフライン動作**のブラウザ 2D CAD。ビルド不要、`index.html` を開くだけで動作する。

## 設計方針

- **バニラ HTML + CSS + JavaScript**（ビルドツール不要、CDN 非依存、外部通信ゼロ）
- **AutoCAD と同じ操作モデル**
  - 画面下部のコマンドラインが主役。`LINE` / `L`、`CIRCLE` / `C` などのコマンドとエイリアス
  - 座標入力: 絶対 `10,20` / 相対 `@5,5` / 極座標 `@10<45` / 直接距離入力（数値のみ）
  - Enter / Space / 右クリック = 確定・コマンド繰り返し、Esc = キャンセル
  - 名詞→動詞（先に選択して MOVE）と 動詞→名詞（MOVE してから選択）の両対応
  - 窓選択（左→右、青 / 内包）・交差選択（右→左、緑 / 交差）
  - オブジェクトスナップ（端点・中点・中心・四半円点・交点）、直交モード、極トラッキング、グリッドスナップ
  - F3 / F7 / F8 / F9 / F10 のファンクションキートグル
- **図面データはメモリ上のエンティティ配列**。DXF へシリアライズして保存

## ファイル互換性

| 形式 | 読み込み | 書き出し | 備考 |
|------|---------|---------|------|
| DXF (ASCII, R12〜2018) | ✅ 対応エンティティのみ | ✅ R12 形式で出力 | LINE / CIRCLE / ARC / LWPOLYLINE / POLYLINE / TEXT / MTEXT(簡易) / POINT、LAYER テーブル、bulge(円弧付きポリライン)。**寸法は線分+文字に分解して出力**、画像・拘束は含まれない |
| JSON プロジェクト | ✅ | ✅ | 自形式。**寸法・画像（dataURL）・拘束・寸法スタイルなど DXF R12 で保持しにくい情報を完全保存**。SAVEJSON / OPENJSON |
| DWG | ❌ | ❌ | **DWG はオートデスク非公開のバイナリ形式のため、純ブラウザ・オフラインでの完全互換は不可能。** 将来 LibreDWG (WASM) の同梱を検討（Phase 7）。当面は AutoCAD / 無償の ODA File Converter 等で DXF に変換して使用する |

> 書き出しを R12 にしている理由: ハンドルテーブル等の必須ボイラープレートが最小で、
> AutoCAD 本体・互換 CAD・ビューアの実装がもっとも広く受け付ける「最大公約数」形式のため。

## 実装済みコマンド（v1）

### 作図
| コマンド | エイリアス | 内容 |
|---|---|---|
| LINE | L | 線分（連続入力、U で1点戻る、C で閉じる） |
| CIRCLE | C | 円（中心→半径 or 直径 D） |
| ARC | A | 円弧（3点） |
| PLINE | PL | ポリライン（C で閉じる） |
| RECTANG | REC | 長方形（2対角点） |
| POINT | PO | 点 |
| TEXT | T, DT | 文字（挿入点→高さ→角度→文字列） |

### 修正
| コマンド | エイリアス | 内容 |
|---|---|---|
| MOVE | M | 移動 |
| COPY | CO, CP | 複写（連続配置） |
| ROTATE | RO | 回転（角度入力 or 点指定） |
| SCALE | SC | 尺度変更 |
| MIRROR | MI | 鏡像（元を残す/削除を選択） |
| OFFSET | O | オフセット（線分・円・円弧） |
| ERASE | E | 削除（Delete キーでも可） |
| EXPLODE | X | ポリライン・寸法の分解 |
| TRIM | TR | トリム（切り取りエッジ選択 or Enter=全図形、線分・円・円弧対応） |
| EXTEND | EX | 延長（境界エッジまで、線分・円弧対応） |
| FILLET | F | フィレット（R キーワードで半径変更、R=0 でコーナー） |
| CHAMFER | CHA | 面取り（D キーワードで距離変更） |

### 寸法（v2）
| コマンド | エイリアス | 内容 |
|---|---|---|
| DIMLINEAR | DLI | 長さ寸法（配置位置で水平 / 垂直を自動判定） |
| DIMALIGNED | DAL | 平行寸法 |
| DIMRADIUS | DRA | 半径寸法（R 表記） |
| DIMDIAMETER | DDI | 直径寸法（⌀ 表記） |
| DIMSTYLE | D | 寸法スタイル管理（文字高さ・矢印・精度・尺度など、作成 / 変更 / 適用） |

### パラメトリック拘束（v2）
| コマンド | エイリアス | 内容 |
|---|---|---|
| GCCOINCIDENT ほか | GCC/GCL/GCN/GCP/GCE/GCH/GCV/GCT/GCS/GCQ/GCM/GCF | 幾何拘束: 一致 / 同一線上 / 同心円 / 平行 / 直交 / 水平 / 垂直 / 接線 / 対称 / 等値 / 中点 / 固定 |
| DCHORIZONTAL ほか | DCH/DCV/DCA/DCANG/DCR/DCD | 寸法拘束: 水平距離 / 垂直距離 / 平行距離 / 角度 / 半径 / 直径（値の編集で図形が追従） |
| CONSOLVE | RESOLVE | 拘束の再計算 |
| CONDELETE | DELCON | 選択オブジェクトの拘束を削除 |

数値ソルバー（Levenberg-Marquardt + ヤコビアンのランク解析）により、
**過拘束（冗長）／矛盾拘束／未拘束自由度** をサイドバーの拘束パネルに表示する。

### 画像・プロジェクト（v2）

## 使い方

1. `index.html` をブラウザで開く（サーバ不要、file:// で動作）
2. コマンドラインに `L` と入力して Enter → 始点・次点をクリック → Enter で終了
3. 保存はリボンの「DXF 書き出し」または `SAVEAS`

## 既知の制約（v2）

- DWG 読み書き不可（上記の通り。DXF 経由で運用）
- HATCH / BLOCK / STRETCH / ARRAY / グリップ編集は未実装（ロードマップ参照 → claude.md）
- TRIM / EXTEND の対象は線分・円・円弧（ポリラインは EXPLODE してから）。FILLET / CHAMFER は線分同士
- 寸法は非自己記述型（作成時の点・半径のスナップショット。図形変更に自動追従しない）。追従が必要な場合は寸法拘束を使用
- 拘束の対象は線分・円・円弧・点（ポリライン・文字・寸法・画像には設定不可）
- DXF の INSERT（ブロック参照）・HATCH・SPLINE・楕円・DIMENSION は読み込み時スキップ（件数を通知）
- 線種は CONTINUOUS / DASHED / CENTER / HIDDEN の4種
- 印刷（PLOT）・ペーパー空間・レイアウトタブは未実装
